This code takes the datasets Snapshot Europe 2021 (Iannarilli et al. 2025a) and Snapshot Europe 2022 (Iannarilli et al. 2025b) archived in Wildlife Insights, runs a few data-cleaning steps, and compiles the two datasets into one to produce the data presented in Iannarilli et al. 2026. It also produces figures, tables and summaries provided in the Metadata file of the related  publication.
